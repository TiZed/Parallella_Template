Parallella Board - Supercomputing for Everyone

Expansion Board

Project template for interfacing with the Parallella board.

This base project includes a schematic hierarchical block with the four PEC connectors and the associate signals with each connector. The Samtec BTH connectors foot prints are included and placed aligned to the Parallella board. The board edges are marked as a comment for convenience.
